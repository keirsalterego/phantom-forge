## Application Strategy

### Phase 1 (Month 1 — While Building Portfolio)
- Apply to: 50 companies (mostly startups + mid-tier)
- Focus: Get interview practice, refine pitch
- Priority: Roles matching your current skills (Python scripting, basic Rust)

### Phase 2 (Month 2 — Growing Skills)
- Apply to: 200 companies (startups + big tech)
- Focus: Technical interviews intensifying
- Priority: Roles matching intermediate Rust/Python skills

### Phase 3 (Month 3 — Senior Push)
- Apply to: 250+ companies (all tiers including FAANG/MANGA)
- Focus: Negotiate multiple offers
- Priority: Senior roles you now qualify for
