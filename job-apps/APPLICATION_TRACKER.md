## Response Rate Targets

| Metric | Target |
|--------|--------|
| Applications → Phone Screen | 10-15% |
| Phone Screen → Technical | 50-60% |
| Technical → Onsite | 40-50% |
| Onsite → Offer | 30-40% |
| Overall conversion | 1.5-3% |

If your overall conversion is below 1%, your resume needs work.
If you're getting screens but bombing technicals, study harder.
